### Overview

#### Installation

Rubygems:

    $ gem install gamebox

From source:

    $ git clone git://github.com/shawn42/gamebox.git
    $ cd gamebox
    $ bundle install
    $ rake install

#### Game Creation

To create a new Gamebox project run:

    $ gamebox new zapper

To run your game:

    $ cd zapper
    $ bundle
    $ bundle exec rake
